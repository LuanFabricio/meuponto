# Meu Ponto

Um aplicativo simples para gerenciamento de ponto do trabalhador.

## Features
- Adicionar turnos padrão do expediente de trabalho.
- Gerenciar horario de entrada e saída do dia.
- Ver turnos dos últimos dias
